# SOS Mulher Brasil

Aplicativo Android desenvolvido para auxiliar mulheres em situação de risco, oferecendo acesso rápido a serviços de emergência e informações importantes.

## Funcionalidades
- Contatos de emergência (190, 180, 192)
- Localização de pontos de apoio
- Informações sobre direitos da mulher

## Tecnologias
- Kotlin
- Android Studio
- XML
